Set up Claude grading (1 of 7)

Claude reads your guesses about the hidden hands and explains where your reasoning is sharp and where it's off. About a penny per deal. You can turn it off any time in settings.

To use it, you'll set up an account at Anthropic (the company that makes Claude), put a little money on it ($5 is enough), and copy a special password — an API key — into this trainer. The key lives only in this browser; we never see it.

Already a Claude Code or Claude.ai subscriber? You still need to do this. Subscriptions cover Claude.ai and Claude Code, but not using Claude in other apps like this one — that's billed separately, from API credit.

The next six steps walk you through it. Take your time. You can use the Back button if you lose your place.

Sign in to Anthropic (2 of 7)

Click the button below — it opens Anthropic's website in a new tab.

If you have a Claude.ai or Claude Code account already: sign in with the same email and password. Anthropic uses one account for everything.

If this is your first time:

  1. Click Sign Up.
  2. Type your email address and click Continue with Email.
  3. Anthropic emails you a 6-digit code. Open your email, copy the code, paste it back into the Anthropic page.
  4. It may ask for your name, phone number, and an organization name — you can put anything (e.g. "Personal"). Click through.

Once you're signed in, come back here and click Next.

Add $5 of API credit (3 of 7)

The Anthropic API uses prepaid credit. You put a dollar amount on, and each Claude question deducts a tiny fraction of a cent. $5 is enough for hundreds of trainer deals.

On the Anthropic Console:

  1. Click Get API Key (large button on the dashboard) — or in the left sidebar, click Plans & Billing and then Add Credits.
  2. A panel titled "Start building with Claude" appears, asking how much credit to add. The default is $50 — that's more than you need.
  3. Click the Other box (top right of the four options) and type 5 for $5.
  4. Fill in your billing address (your name will already be there).
  5. Click Credit card to expand it. Enter your card number, expiration date, and CVV.
  6. Leave "shipping address is the same as billing address" checked.
  7. Scroll down and click Submit (or Pay).

You should see $5.00 available credit on the dashboard once it goes through. Then click Next.

Set a spending limit (4 of 7) — optional

This is a safety net so the trainer can't go above what you've set, even if something runs in a loop by mistake.

  1. In the left sidebar, click Plans & Billing.
  2. Look for Spend limits (sometimes called Usage limits).
  3. Set a monthly limit of $5 and save.

If you can't find this setting, that's OK — skip it. The $5 of credit you added is already a hard ceiling (the API can't spend more than what you've loaded). Click Next.

Create your API key (5 of 7)

The API key is a long password the trainer uses to ask Claude questions. You'll create one now and paste it on the next screen.

  1. In the left sidebar of the Anthropic Console, click API Keys.
  2. Click the Create Key button (top right of the page).
  3. Give it a name like Bridge Trainer so you'll recognize it later. Leave other options at their defaults. Click Create Key.
  4. A long string appears that starts with sk-ant-. That's your key.
  5. Important: click the Copy button next to the key. Anthropic only shows it once. If you close the dialog without copying, you'll have to create another key.

Once the key is on your clipboard, click Next.

Paste your key (6 of 7)

Click in the box below and paste your key (⌘V on Mac, Ctrl-V on Windows, or right-click and choose Paste). It should start with sk-ant-.

Then click Test & Save. The trainer makes one tiny call to Anthropic to confirm the key works (costs a fraction of a cent).

All set (7 of 7)

Your key is saved in this browser and Claude grading is on. Pick a scenario and play a few tricks — by default, the grading panel pops up after trick 3 and asks for your read on the hidden hands.

You can change when grading appears, or remove the key, any time from the gear icon in the top right.
